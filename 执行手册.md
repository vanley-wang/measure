# 类器官分割量化项目 — 完整执行手册

> 适用场景：基于 **nnUNet 分割结果** 的类器官三维量化、聚类分型、PCA 综合评分、药效关联分析

> 数据格式：`.nii.gz`（nnUNet 推理输出）→ `.mat` → `.xlsx`

## 二、完整执行流程

### Step 0：环境准备（首次运行时）

```bash
pip install nibabel numpy pandas scipy scikit-image openpyxl tqdm scikit-learn
```

---

### Step 1：格式转换（`.nii.gz` → `.mat`）

**脚本**：`nnunet_bridge.py`

**功能**：
- 读取 nnUNet 预测 mask + 原始 OCT 图像
- 对整孔 mask 做连通域标记（每个类器官一个 ID）
- 输出 `.mat` 文件，兼容现有量化代码

**输入**：
```
nnUNet_Data/FXN_0701/organoid_001_0000.nii.gz      # 原始图
nnUNet_Data/FXN_0701_prediction/organoid_001.nii.gz # 预测mask
```

**输出**：
```
Data/nnUNet_FXN/FXN_0701/
    ├── seg_fill/B2_0701_fill.mat      # 二值填充掩膜 (Data_fill)
    ├── seg_label/B2_0701_label.mat    # 实例标签 (Data_label)
    └── scatt_mat/B2_0701_scatt.mat    # OCT原始强度 (data_scatt)
```

**命令**：
```bash
python nnunet_bridge.py
```

**注意事项**：
- Windows 下多进程可能报错，若出错改 `N_WORKERS = 1`
- B2_0701 原始图可能损坏（gzip 解压失败），会自动跳过

---

### Step 2：提取形态学特征

**脚本**：`measure_from_label.py`

**功能**：
- 从 `seg_label/*.mat` 读取每个类器官的实例标签
- 提取 9 个形态学特征：体积、表面积、囊腔体积、囊腔数、长轴、短轴、壁厚、球形度
- 输出到 `measure_excel/*.xlsx`

**输入**：`Data/nnUNet_FXN/FXN_0701/seg_label/*_label.mat`

**输出**：`Data/nnUNet_FXN/FXN_0701/measure_excel/B2_0701.xlsx`

| 列名 | 含义 |
|------|------|
| Index | 唯一标识 (如 B2_0701_1) |
| Organoids_Volume | 体积（像素数） |
| Organoids_Volume_Fill | 填充后体积 |
| Organoids_Surface | 表面积 proxy |
| Cavity_Volume | 囊腔体积 proxy |
| CavityNum | 囊腔数 |
| LongAxis | 长轴长度 |
| ShortAxis | 短轴长度 |
| Wall_Thickness | 壁厚 proxy |
| Sphericity | 球形度 proxy |
| Scatt_Mean | **占位（NaN），Step 3 填充** |
| Scatt_STD | **占位（NaN），Step 3 填充** |

**命令**：
```bash
python measure_from_label.py
```

**依赖**：Step 1（`seg_label` 必须已生成）

---

### Step 3：合并散射统计

**脚本**：`scatt.py`

**功能**：
- 读取 `seg_label`（每个 organoid 的 mask）+ `scatt_mat`（OCT 强度）
- 统计每个 organoid 区域内的散射均值/标准差
- 合并到 `measure_excel/*.xlsx`

**输入**：
```
Data/nnUNet_FXN/FXN_0701/seg_label/*_label.mat
Data/nnUNet_FXN/FXN_0701/scatt_mat/*_scatt.mat
```

**输出**：更新 `measure_excel/*.xlsx`（填充 Scatt_Mean / Scatt_STD）

**命令**：
```bash
python scatt.py
```

**依赖**：Step 1 + Step 2

**注意事项**：
- Windows 内存限制，建议 `max_workers=1`
- B2_0701 无 `scatt_mat`，Scatt 列保持 NaN

---

### Step 4：聚类分型（4 类表型）

**脚本**：`cluster-merge.py`

**功能**：
- 加载基于 nnUNet 数据重新训练的 KMeans 模型（只用 9 个形态学特征）
- 对每个 organoid 预测聚类标签（0~3）
- 映射为生物学表型名称

**模型**：
```
model/Kmeans-nnUNet-morph.pickle     # KMeans 模型（9维）
model/scaler-nnUNet-morph.pickle     # StandardScaler
```

**输出**：`Data/nnUNet_FXN/FXN_0701/cluster_merge/B2_0701_merge.xlsx`

| Cluster | 表型 | 特征 |
|---------|------|------|
| 0 | 小体积型 | 体积小、细胞少 |
| 1 | 巨大囊泡型 | 体积极大、有囊腔 |
| 2 | 中等体积型 | 体积中等、最常见 |
| 3 | 大体积型 | 体积大、结构致密 |

**命令**：
```bash
python cluster-merge.py
```

**依赖**：Step 2 + Step 3（需要完整 measure_excel）

---

### Step 5：孔板级汇总（各表型群体统计）

**脚本**：`cluster_analysis_1.py`

**功能**：
- 读取 `cluster_merge/*.xlsx`
- 按 4 个 Cluster 分别统计：数量、平均体积、平均表面积、总囊腔体积、平均长/短轴、平均壁厚、球形度、散射均值/标准差
- 在原 Excel 中追加 `Sheet2`

**输入**：`Data/nnUNet_FXN/FXN_0701/cluster_merge/*.xlsx`

**输出**：每个 `.xlsx` 追加 `Sheet2`

**命令**：
```bash
python cluster_analysis_1.py
```

**依赖**：Step 4

---

### Step 6：汇总为主表

**脚本**：`merge_analysis.py`

**功能**：
- 读取所有 `cluster_merge/*.xlsx` 的 `Sheet2`
- 合并为一张大表

**输出**：`Data/nnUNet_FXN/nnUNet_Analysis.xlsx`

**命令**：
```bash
python merge_analysis.py
```

**依赖**：Step 5

---

### Step 7：粗糙度计算（可选）

**脚本**：`roughness.py`

**功能**：
- 基于 3D Butterworth 滤波 + FFT 功率谱计算每个类器官的表面粗糙度
- 只计算 Cluster 0 和 1（可根据需要调整 TARGET_CLUSTERS）

**输入**：`seg_fill/*.mat` + `cluster_excel/*.xlsx`

**输出**：`Data/nnUNet_FXN/FXN_0701/roughness/*.xlsx`

**命令**：
```bash
python roughness.py
```

**依赖**：Step 1（seg_fill）+ Step 4（cluster）

---

### Step 8：粗糙度汇总到主表（可选）

**脚本**：`roughness_analysis.py`

**功能**：
- 读取 `roughness/*.xlsx`
- 汇总 Roughness_All、Roughness_Avg_1、Roughness_Avg_2 到主表

**输入**：`nnUNet_Analysis.xlsx` + `roughness/*.xlsx`

**输出**：更新 `nnUNet_Analysis.xlsx`

**命令**：
```bash
python roughness_analysis.py
```

**依赖**：Step 6 + Step 7

---

### Step 9：PCA 综合评分

**脚本**：`PCA.py`

**功能**：
- 对 `nnUNet_Analysis.xlsx` 做 PCA 降维
- 输出特征根、载荷矩阵、最终评分系数

**输入**：`Data/nnUNet_FXN/nnUNet_Analysis.xlsx`

**输出**：`Data/nnUNet_FXN/PCA_Model_Details.xlsx`

包含 4 个 Sheet：
1. `1_Eigenvalues` — 方差解释率
2. `2_Loadings_Matrix` — 载荷矩阵
3. `3_Final_Coefficients` — 各特征对综合评分的贡献权重
4. `4_Feature_Meta` — 特征分组映射

**命令**：
```bash
python PCA.py
```

**依赖**：Step 6（或 Step 8）

**输出**：`Data/nnUNet_FXN/nnUNet_PCA_Scores.xlsx`

| PC2 | 第2主成分得分 |
| PC3 | 第3主成分得分 |
| PC4 | 第4主成分得分 |
| Result | 加权综合评分 |

**命令**：
```bash

```

**依赖**：Step 9

## 六、常见问题速查

### Q1：B2_0701 没有 Scatt 数据怎么办？
**A**：原始图 `organoid_001_0000.nii.gz` gzip 损坏。该孔 Scatt 列为 NaN，不影响形态学特征。

### Q2：scatt.py 内存不足报错？
**A**：改 `max_workers=1`，或优化 `extract_scatt_stats_fast` 用 `np.bincount`。

### Q3：聚类模型维度不匹配？
**A**：确保使用 `model/Kmeans-nnUNet-morph.pickle`（9 维形态特征），而非旧的 `Kmeans-scatt.pickle`。

### Q4：nnUNet Scatt 值比 SAM2UNet 小很多？
**A**：正常。nnUNet `.nii.gz` 预处理压缩了强度范围。绝对值不可跨模型对比，模型内相对比较仍有效。

### Q5：PCA Score 正负值什么意思？
**A**：Result = w1*PC1 + w2*PC2 + w3*PC3 + w4*PC4。正值表示该孔特征与 PC1 方向一致，负值相反。
